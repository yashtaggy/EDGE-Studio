# **App Name**: EDGE Central Command Portal

## Core Features:

- Centralized Navigation Dashboard: Displays four large, responsive navigation buttons in a 2x2 grid, each linked to a specific internal resource.
- External Link Integration: Each navigation button securely opens a specified external URL in a new browser tab with appropriate security attributes ('target='_blank'', 'rel='noopener noreferrer'').
- Responsive User Interface: Ensures the application's layout and components adapt gracefully across various screen sizes, from desktop to mobile devices.
- AI Chatbot Widget Placeholder: Reserves a dedicated visual space on the main interface for future integration of an AI chatbot widget, without current AI logic.
- AI Data Ingestion API Placeholder: Implements a placeholder API endpoint ('/api/agent/ingestData') that will, in the future, be used by an AI tool for data ingestion, currently returning 'AI integration pending; logic not implemented.'
- AI Query API Placeholder: Implements a placeholder API endpoint ('/api/agent/query') designed for future use by an AI tool to process queries, currently returning 'AI integration pending; logic not implemented.'
- Secure URL Handling: Implements sanitization for all URL values prior to rendering to prevent injection vulnerabilities.

## Style Guidelines:

- Primary Color: A sophisticated red-orange (#FF5D4E) evoking professionalism and trust, suitable for button backgrounds and key interface elements.
- Background Color: A black (#000000) providing a clean, corporate backdrop for enhanced readability.
- Accent Color: A refreshing moderate white (#ECF6F8) used sparingly for highlights or interactive states to draw attention without being overwhelming.
- All text: 'ES Klarheit Sans' for its modern, clean, and highly legible appearance, consistent with a corporate aesthetic.
- Page Title: 'EDGE Central Command Portal' prominently displayed at the top of the interface.
- Button Grid: Four Flashcards arranged in a clean, balanced grid.
- Buttons should utilize simple, modern icons where appropriate to visually communicate their function, aligned with a corporate, minimalist design.
